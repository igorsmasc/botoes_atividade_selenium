LIVRE

O seletor por ID é utilizado para selecionar um elemento HTML pelo seu ID único. Para utilizar esse seletor, utilize o caractere '#' seguido pelo ID do elemento que deseja selecionar. Exemplo: #meu-elemento

O seletor por CSS é utilizado para selecionar elementos HTML com base em suas propriedades CSS. Para utilizar esse seletor, utilize a sintaxe CSS. Exemplo: .minha-classe ou tag[atributo=valor]

O seletor por XPath é utilizado para selecionar elementos HTML com base em suas propriedades e localização na árvore DOM. Para utilizar esse seletor, utilize a sintaxe XPath. Exemplo: //div[@class='minha-classe']
