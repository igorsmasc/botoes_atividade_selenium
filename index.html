O seletor ID é usado para selecionar um elemento HTML com base em um ID exclusivo atribuído a ele. Isso é feito utilizando o método findElement(By.id("id-do-elemento")) do WebDriver. O ID é um atributo HTML que é atribuído a um elemento HTML e é exclusivo em toda a página. Por exemplo, <input type="text" id="username"> pode ser selecionado com o seletor ID findElement(By.id("username")).

O seletor ClassName é usado para selecionar um ou mais elementos HTML com base em uma classe específica atribuída a eles. Isso é feito utilizando o método findElement(By.className("nome-da-classe")) do WebDriver. A classe é um atributo HTML que é atribuído a um ou mais elementos HTML. Por exemplo, <button class="botao-2">Enviar</button> pode ser selecionado com o seletor ClassName findElement(By.className("botao-2")).

O seletor Name é usado para selecionar um ou mais elementos HTML com base no valor do atributo "name" atribuído a eles. Isso é feito utilizando o método findElement(By.name("valor-do-atributo-name")) do WebDriver. O atributo "name" é um atributo HTML que é atribuído a um ou mais elementos HTML. Por exemplo, <input type="text" name="usuario"> pode ser selecionado com o seletor Name findElement(By.name("usuario")).

O seletor CSS é usado para selecionar um ou mais elementos HTML com base em um seletor CSS específico. Isso é feito utilizando o método findElement(By.cssSelector("seletor-css")) do WebDriver. Os seletores CSS permitem que você selecione elementos com base em atributos, valores de atributos, classes, IDs e muito mais. Por exemplo, <div class="classe-1"><p>Hello</p></div> pode ser selecionado com o seletor CSS findElement(By.cssSelector("div.classe-1 p")).

O seletor Xpath é usado para selecionar um ou mais elementos HTML com base em um caminho específico no documento HTML. Isso é feito utilizando o método findElement(By.xpath("caminho-xpath")) do WebDriver. Os caminhos XPath permitem que você selecione elementos com base em sua localização no documento HTML, seus atributos e muito mais. Por exemplo, <div class="classe-2"><p>World</p></div> pode ser selecionado com o seletor Xpath findElement(By.xpath("//div[@class='classe-2']/p")).
